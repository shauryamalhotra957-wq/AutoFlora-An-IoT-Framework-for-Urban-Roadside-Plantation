# Contributing

Thanks for improving AutoFlora.

## Development setup

1. Fork or clone the repository.
2. Create a focused branch from `main`.
3. Open `roadside_plant_watering_system.ino` in the Arduino IDE.
4. Select the target board and port, then verify the sketch.

## Validation

Compile for the intended board before opening a pull request. For hardware-facing changes, document the board, sensor versions, wiring assumptions, and observed serial output.

## Pull requests

Keep firmware changes focused. Explain safety implications for pumps or actuators, include test conditions, and never commit private network credentials.
